Usage

A page may contain more than one tab bar.

Tabs

  • Click on a tab to select that tab
  • To close tabs
    • Click on the "×" in the tab to close that tab
    • Middle mouse click anywhere on a tab to close that tab
  • To organize tabs
    • Drag a tab on top of another tab to place the selected tab before it
    • Press "Esc" while moving a tab to cancel moving the tab
    • Move a tab outside of the tab bar and release the mouse button to cancel moving the tab
  • Use the mousewheel to scroll through tabs that overflow the tab bar
  • Double click on an empty area of the tab bar to create a new tab

Tab Settings

  • Movable – This controls the ability to move tabs
  • Closable – This controls the ability to colse tabs
  • Active – This controls whether or not the tab is active

Style

To change the size of the whole tab change the font size on the .tab-bar class as this will scale everything including the tab padding. 

/* Make the tabs font for this tab bar 20px */
#tab-bar-1.tab-bar {
    font-size: 20px;
}

/* Make the tabs font for this tab bar 10px */
#tab-bar-2.tab-bar {
    font-size: 10px;
}
