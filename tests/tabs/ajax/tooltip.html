Tooltip

To add a tooltip to an item add the tooltip option to the Tab constructor.

// Get a tab group
let tabGroup = Workspace.tabGroup('#tab-group-1');

// Add a new tab with a tooltip
tabGroup.addTab(new Workspace.Tab({
    // Set the title of the tab
    title: 'Tool Tip',
    // Set the tooltip to display upon mouseover
    tooltip: 'I am a tooltip'
}));
